(機械翻訳)アプリケーションの配布¶

アプリケーションが正常に動作したら、 `` setup.py sdist``コマンドを使って&quot;tarball &quot;を作成することができます。次のコマンドは、現在の作業ディレクトリに ``チュートリアル ``パッケージと `` setup.py``ファイルがあると仮定しています。

UNIXの場合：

$ $VENV/bin/python setup.py sdist

Windowsの場合：

c:\tutorial> %VENV%\Scripts\python setup.py sdist

そのようなコマンドの出力は次のようになります：

running sdist
# more output
creating dist
Creating tar archive
removing 'tutorial-0.0' (and everything under it)

このコマンドは、 `` tutorial-0.0.tar.gz``という名前の `` dist``サブディレクトリにtarballを作成します。このファイルを友達に送信して、クールな新しいアプリケーションを表示することができます。彼らは `` pip install``コマンドを直接指してインストールすることができます。あるいは、あなたは PyPI <https://pypi.org/> `_これを他の世界と共有します。PyPIから他のパッケージの人々のダウンロードと同様に、 `pip install``を介してダウンロードできます。
