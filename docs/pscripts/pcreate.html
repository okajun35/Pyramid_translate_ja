(機械翻訳) PCreate¶

ピラミッドの足場を出力ディレクトリに描画します。

注：Pyramid 1.8以降、このコマンドは推奨されていません。代わりに特定のcookiecutterを使用してください：https://github.com/Pylons/?q=cookiecutter

usage: pcreate [-h] [-s SCAFFOLD_NAME] [-t SCAFFOLD_NAME] [-l]
               [--list-templates] [--package-name PACKAGE_NAME] [--simulate]
               [--overwrite] [--interactive] [--ignore-conflicting-name]
               [output_directory]
output_directory¶

プロジェクトが作成されるディレクトリです。

-h, --help¶

このヘルプメッセージを表示して終了する

-s <scaffold_name>, --scaffold <scaffold_name>¶

作成プロセスに足場を追加する（multiple -s args accepted）

-t <scaffold_name>, --template <scaffold_name>¶

-s / - scaffoldの下位互換性の別名です。作成プロセスに足場を追加する（複数の-t argsが受け入れられる）

-l, --list¶

すべての利用可能なスカフォールド名をリストする

--list-templates¶

-l / - listの後方互換性の別名。利用可能な足場の名前をすべて列挙してください。

--package-name <package_name>¶

使用するパッケージ名。提供された名前は有効なPythonパッケージ名とみなされ、検証されません。デフォルトでは、パッケージ名はoutput_directoryの値から導出されます。

--simulate¶

シミュレートするが動作しない

--overwrite¶

常に上書きする

--interactive¶

ファイルが上書きされるときに問い合わせます（これがデフォルトですが、上書きするように指定することができます--overwrite）

--ignore-conflicting-name¶

選択した名前が既に存在する/インポート可能なパッケージの名前であっても、プロジェクトを作成します。
